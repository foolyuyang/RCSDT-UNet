# RCSDT-UNet
RCSDT-UNet for ONFH Diagnosis and Segmentation

The RCSDT-UNet.ipynb file carried out ONFH ROI segmentation and dignosis from the "input_data" directory, and the results would be shown in "output_mask" directory.

The trained model weight file "best_model.pth" exceeded the limit of github respository, and could be downloaded from the following link: https://drive.google.com/file/d/13bY_DsRwsjOkCBmlqaKt3J4zxRrjOc1u/view?usp=sharing.

The demos of the input_data included ONFH cases randomly selected from the publicly available JLU-ONFH Dataset with link: https://github.com/sole12343/JLU-ONFH-Dataset, which was introduced in paper with citation:
Shen X, Luo J, Tang X, Chen B, Qin Y, Zhou Y, Xiao J. Deep Learning Approach for Diagnosing Early Osteonecrosis of the Femoral Head Based on Magnetic Resonance Imaging. J Arthroplasty. 2022 Oct 12:S0883-5403(22)00900-7. doi: 10.1016/j.arth.2022.10.003. Epub ahead of print. PMID: 36243276.
